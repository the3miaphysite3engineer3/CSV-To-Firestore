Simple Streamlit app to explore CSV files in this folder.

Quick start

1. Create a virtual environment (recommended):

```powershell
python -m venv .venv
.\.venv\Scripts\activate
```

2. Install dependencies:

```powershell
pip install -r requirements.txt
```

3. Run the app:

```powershell
streamlit run app.py
```

The app will list CSV files in the project folder, let you preview data, show summary statistics, and draw a quick scatter plot of numeric columns.
